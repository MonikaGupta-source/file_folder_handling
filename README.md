# 📁 File & Folder Manager

A terminal-based File and Folder Management System built using **Python** — perform all CRUD operations on files and folders directly from the terminal!

---

## ✨ Features

**📂 Folder Operations:**
- 🆕 Create a new folder
- 📋 Read / list all folders & files
- ✏️ Rename a folder
- ❌ Delete a folder

**📄 File Operations:**
- 🆕 Create a new file with content
- 📖 Read file content
- ✏️ Update a file — rename, append, or overwrite
- ❌ Delete a file



## 💡 How It Works

```
1. Create folder       5. Create file
2. Read folders        6. Read file
3. Update folder       7. Update file
4. Delete folder       8. Delete file
0. Exit
```
